# space_invaders_project
Porting a space invaders game to different platforms and frameworks

### Platforms
- Android
  - Godot
  - MonoGame

- iOS
  - MonoGame

- PC (Windows (Win32/UWP) / Linux / Mac)
  - MonoGame (Windows 10 UWP)
  - SDL2
  - GLFW3
  - Godot

- Xbox One
  - MonoGame (UWP)

- GameBoy Advanced
- MS-DOS
- Sega Dreamcast
- Sega Saturn
- PlayStation Portable

- HTML5
  - Godot
  
- G-eon

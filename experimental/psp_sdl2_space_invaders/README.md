# psp-sdl2-space-invaders

Space Invaders written in C++ using SDL2

Built with this guide [pspdev](https://pspdev.github.io/)

#### Note: This is experimental

- Runs unstable (with glitches) on PPSSPP
- Does not boot on PSP 3004
